## 5. 接口测试示例

### 5.1 用户登录接口

**请求：**
```http
POST /api/auth/login/
Content-Type: application/json

{
    "username": "admin",
    "password": "admin123",
    "captcha": "abcd",
    "captcha_key": "xxx"
}
```

**成功响应：**
```http
HTTP/1.1 200 OK
Content-Type: application/json

{
    "success": true,
    "code": 200,
    "message": "登录成功",
    "data": {
        "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
        "user_info": {
            "id": 1,
            "username": "admin",
            "email": "admin@example.com",
            "real_name": "管理员"

    }
}
```
